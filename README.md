Vous êtes un nouveau fermier qui venez d'acheter d'un drone pour controler votre champ à distance.
Votre but, vous développer dans votre metier en vendant des légumes et achetez des améliorations.
Ce but sera atteint qaund vous aurez un score de 2 500 000 points.

Controles (Clavier Souris):
ZQSD : Movement
Clic gauche : Interagir
Clic Droit : Arroser
A : Magasin
E : Mode Blueprint (après avoir acheter le premier drone)
Echap : Menu Pause

Controles (Manette):
Stick Gauche: Mouvement
A : Interagir
Y : Arroser
X : Magasin
B : Mode Blueprint (après avoir acheter le premier drone)
+ : Menu Pause

Ceci est un jeu réalisé pour ma 3ème année de BUT Informatique.
